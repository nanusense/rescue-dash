# Snake Rescue Archive — shiftingradius.com

A static dashboard for all 372+ rescue entries. Reads from `rescue-data.json` — no backend, no live API calls.

---

## File structure

```
your-repo/
├── index.jsx              ← the dashboard (deploy via GitHub Pages)
├── rescue-data.json       ← all rescues, generated by the script
├── update-rescues.mjs     ← run this when you add new rescues
└── README.md
```

---

## First-time setup

### 1. Clone / create repo on GitHub
Create a new repo (e.g. `rescue-archive`), clone it locally, drop these files in.

### 2. Set your Anthropic API key
```bash
export ANTHROPIC_API_KEY=sk-ant-your-key-here
```

### 3. Run the full build (one time only)
```bash
node update-rescues.mjs
```
This fetches all 10 pages, parses every rescue via Claude, writes `rescue-data.json`,
and commits + pushes to GitHub. Takes ~3–4 minutes. Costs ~$0.20.

### 4. Enable GitHub Pages
- Go to repo Settings → Pages
- Set source to `main` branch, root `/`
- GitHub gives you a URL like `https://yourusername.github.io/rescue-archive`

### 5. Point your subdomain
In your domain registrar (or Cloudflare), add a CNAME:
```
rescues.shiftingradius.com  →  yourusername.github.io
```
In the GitHub repo, add a file called `CNAME` containing:
```
rescues.shiftingradius.com
```
GitHub Pages handles HTTPS automatically.

## Adding new rescues (incremental update)

When you've added new entries to the current year's page on shiftingradius.com:

```bash
export ANTHROPIC_API_KEY=sk-ant-your-key-here
node update-rescues.mjs --update
```

- Only fetches the current year's page (sr-26 etc.)
- Merges new rescues into existing JSON
- Commits and pushes automatically
- Costs ~$0.02 per run

---

## Switching to a new year

When you start a new year page (e.g. sr-27):

1. Open `update-rescues.mjs`
2. In the `SOURCES` array, set `current: false` on sr-26
3. Add a new entry for sr-27 with `current: true`
4. Run `node update-rescues.mjs --update`

## Local development

The dashboard is a React JSX file. To run locally:

```bash
npm create vite@latest rescue-dash -- --template react
cd rescue-dash
# replace src/App.jsx with index.jsx
# place rescue-data.json in /public/
npm run dev
```

Or just open it directly on GitHub Pages — that works fine for viewing.
